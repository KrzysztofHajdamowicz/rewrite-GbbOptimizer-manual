# Tuya Integration

## Getting API Credentials

1. Create an account at [platform.tuya.com](https://platform.tuya.com/)
2. Go to **Cloud → Development**
3. Click **"Create Cloud Project"**
4. Select the **Data Center closest to your location**
5. Click **"Authorize"** on the next screen

This generates:
- **Access ID** (Client ID)
- **Access Secret** (Client Secret)

Enter these credentials in GbbOptimizer's plant configuration to enable Tuya integration.
