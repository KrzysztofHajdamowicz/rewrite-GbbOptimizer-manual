# DeyeCloud Connection

{{< badge "recommended" >}}

DeyeCloud is the recommended cloud connection method for Deye inverters, replacing Solarman due to its improved reliability.

## Setup

1. Create a DeyeCloud account at the DeyeCloud portal
2. In GbbOptimizer, create a new plant or change your existing plant type to **DeyeCloud**
3. Enter your DeyeCloud account credentials
4. Select your inverter device

## Advantages over Solarman

- More reliable connection
- Better API stability
- Can be added as a backup connection alongside other methods
