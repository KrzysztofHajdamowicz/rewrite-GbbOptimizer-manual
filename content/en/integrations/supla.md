# Integration with Supla

GbbOptimizer can control [Supla](https://www.supla.org/) devices (e.g. relays, smart sockets) via Direct Links.

## Configuration

To connect a Supla device to GbbOptimizer, create a **Direct Link** in the Supla Cloud panel:

1. Log in to [cloud.supla.org](https://cloud.supla.org/)
2. Go to the device you want to control
3. Create a Direct Link with the appropriate permissions (on/off)
4. Copy the generated URL
5. Paste the URL in the GbbOptimizer configuration under **IoT** -> **Supla**

> [!NOTE]
> A detailed description of creating Direct Links can be found in the [Supla documentation](https://gui-generic-builder.supla.io/guide/funkcjonalno%C5%9Bci%20supla/sterowanie-linki-bezposrednie).
