# DeyeCloud

{{< badge "deye-only" >}}

DeyeCloud is een back-up-verbindingsmethode die beschikbaar is voor installaties van het type **Solarman**. Kan worden gebruikt als back-up bij problemen met Solarman of als enige verbinding.

## Werkingsmodi

| Modus | Beschrijving |
|------|------|
| **Uitgeschakeld** | DeyeCloud wordt niet gebruikt |
| **Ingeschakeld** | DeyeCloud wordt automatisch geactiveerd wanneer Solarman een fout meldt |
| **Alleen back-up** | Gebruik Solarman niet — altijd DeyeCloud |

## Configuratie

1. Kies in de sectie **Back-upverbinding — DeyeCloud** van de installatieparameters de werkingsmodus
2. Voer de **inloggegevens** voor DeyeCloud in (kunnen afwijken van die voor Solarman)
3. Vink **Inloggegevens onthouden** aan — zonder dit is handmatig opnieuw inloggen vereist
4. Kies na verbinding het **SerialNo van de omvormer**

Een gedetailleerde beschrijving van de parameters vind je in [Installatieparameters]({{< relref "/installation/installation-parameters" >}}).

> [!NOTE]
> DeyeCloud en Solarman zijn twee onafhankelijke portalen met afzonderlijke inloggegevens. Zorg dat je een account hebt op beide diensten als je een back-upverbinding wilt gebruiken.
