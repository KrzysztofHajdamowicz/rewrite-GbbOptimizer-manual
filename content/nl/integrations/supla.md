# Integratie met Supla

GbbOptimizer kan [Supla](https://www.supla.org/)-apparaten (bijv. relais, stopcontacten) aansturen via directe links (Direct Links).

## Configuratie

Om een Supla-apparaat met GbbOptimizer te verbinden, moet je een **directe link** aanmaken in het Supla Cloud-paneel:

1. Log in op [cloud.supla.org](https://cloud.supla.org/)
2. Ga naar het apparaat dat je wilt aansturen
3. Maak een directe link (Direct Link) aan met de juiste rechten (aan/uit)
4. Kopieer de gegenereerde URL
5. Plak de URL in de GbbOptimizer-configuratie onder **IoT** -> **Supla**

> [!NOTE]
> Een gedetailleerde beschrijving van het aanmaken van directe links vind je in de [Supla-documentatie](https://gui-generic-builder.supla.io/guide/funkcjonalno%C5%9Bci%20supla/sterowanie-linki-bezposrednie).
