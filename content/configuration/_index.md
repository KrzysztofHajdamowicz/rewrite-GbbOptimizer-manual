# Configuration

GbbOptimizer is organized into modules, each controlling a specific aspect of your energy optimization. This section covers all configurable modules.

| Module | Purpose |
|--------|---------|
| [Plant Parameters]({{< relref "plant-parameters" >}}) | Core system settings — battery, location, grid limits |
| [Battery Forecast]({{< relref "battery-forecast" >}}) | Forecasting engine and test mode |
| [Charge]({{< relref "charge" >}}) | Battery charging schedules and SOC limits |
| [Discharge]({{< relref "discharge" >}}) | GridSetpoint, MinSOC, Dynamic Discharge |
| [Prices]({{< relref "prices" >}}) | Purchase/sell price formulas |
| [Profiles of Loads]({{< relref "profiles-of-loads" >}}) | Household consumption patterns |
| [Extra Loads / EV]({{< relref "extra-loads-ev" >}}) | EV charging, heat pumps, AutoConverter |
| [HP / AirCond Forecast]({{< relref "hp-aircond-forecast" >}}) | Heat pump and AC forecasting |
| [Gain / Profit]({{< relref "gain-profit" >}}) | Performance tracking and KPIs |
